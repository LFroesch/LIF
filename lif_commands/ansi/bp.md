# ANSI Boilerplate Code

Basic Colors
30m - Black
31m - Red
32m - Green
33m - Yellow
34m - Blue
35m - Magenta
36m - Cyan
37m - White

Format Modifiers
0; - Normal (no effect)
1; - Bold/Bright
2; - Dim
3; - Italic
4; - Underline
5; - Slow blink
6; - Rapid blink
7; - Reverse/invert
8; - Hidden
9; - Strikethrough
